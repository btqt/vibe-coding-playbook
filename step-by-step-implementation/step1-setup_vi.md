SETUP: Tạo môi trường phát triển được tối ưu hóa cho AI cho [project name] sử dụng [tech stack]

Tôi cần bạn thiết lập một môi trường phát triển hoàn chỉnh giúp tối đa hóa chất lượng tạo code của AI và cung cấp các vòng lặp phản hồi ngay lập tức (immediate feedback loops) để ngăn chặn các lỗi phức hợp.

NGỮ CẢNH DỰ ÁN HIỆN TẠI (CURRENT PROJECT CONTEXT):
- Project type: [web app/API/mobile app/etc.]
- Tech stack: [các frameworks, ngôn ngữ, công cụ cụ thể]
- Team size: [số lượng developers]
- Existing codebase: [dự án mới/dự án hiện có với X dòng code]

CÁC YÊU CẦU ĐỂ TỐI ƯU HÓA AI (REQUIREMENTS FOR AI OPTIMIZATION):

1. HỆ THỐNG PHẢN HỒI NGAY LẬP TỨC (IMMEDIATE FEEDBACK SYSTEMS):
   - Cấu hình TypeScript với các cài đặt nghiêm ngặt nhất có thể
   - Thiết lập ESLint với các quy tắc toàn diện giúp bắt các lỗi phổ biến của AI
   - Cấu hình Prettier để định dạng code nhất quán
   - Thiết lập pre-commit hooks chạy kiểm tra kiểu (type checking) và linting
   - Cấu hình IDE/editor để làm nổi bật lỗi trong thời gian thực

2. HƯỚNG DẪN HÀNH VI CỦA AI (AI BEHAVIOR GUIDELINES):
   - Tạo file .cursor/rules với các tiêu chuẩn phát triển rõ ràng
   - Bao gồm các pattern cho xử lý lỗi (error handling), cách sử dụng TypeScript, các phương pháp tiếp cận testing
   - Xác định các tiêu chí code review và quality gates
   - Chỉ định các architectural patterns mà AI nên tuân theo

3. QUẢN LÝ NGỮ CẢNH (CONTEXT MANAGEMENT):
   - Thiết lập cấu trúc dự án giúp các pattern dễ khám phá
   - Tạo các triển khai mẫu (example implementations) cho AI tham khảo
   - Cấu hình tổ chức file giúp nhóm các chức năng liên quan
   - Thiết lập cấu trúc tài liệu cho các quyết định kiến trúc

4. TỰ ĐỘNG HÓA XÁC MINH (VERIFICATION AUTOMATION):
   - Tạo script xác minh chạy sau khi tạo code bởi AI
   - Thiết lập pipeline testing tự động với yêu cầu độ bao phủ cao
   - Cấu hình performance benchmarking cho các critical paths
   - Thiết lập quét security cho các lỗ hổng phổ biến

5. QUY TRÌNH PHÁT TRIỂN (DEVELOPMENT WORKFLOW):
   - Cấu hình các script trong package.json cho các task xác minh AI phổ biến
   - Thiết lập development server với hot reloading
   - Cấu hình các công cụ debugging và logging
   - Thiết lập deployment pipeline với môi trường staging

Vui lòng cung cấp:

1. COMPLETE SETUP SCRIPT: Script duy nhất cấu hình toàn bộ môi trường
2. CONFIGURATION FILES: Tất cả các file config với các cài đặt tối ưu cho phát triển AI
3. VERIFICATION SYSTEM: Script để validate chất lượng code do AI tạo ra
4. WORKFLOW DOCUMENTATION: Quy trình từng bước cho phát triển có sự hỗ trợ của AI

Mục tiêu là một môi trường nơi AI tạo ra code chất lượng cao nhất quán, tích hợp liền mạch với các pattern hiện có và bắt các lỗi ngay lập tức thay vì trong production.

Bắt đầu với setup script và các file cấu hình cốt lõi.
